El presente programa permite llevar un registro de clientes con los datos: alias, nombre, 1er apellido, 2o apellido, razón social, RFC, teléfono y corrreo. Así como realizar operaciones CRUD.
- Está compuesto únicamente por el archivo registroClient.py
- Para compilar y ejecutar el programa basta con escribir en una terminal lo siguiente: python3 registroClient.py

**Nota: es necesario tener instalado python3 en la máquina en la que se pretenda usar el programa.
